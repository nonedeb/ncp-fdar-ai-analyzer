# NCP & FDAR AI Analyzer v1.2

Included:
- NANDA-aware and FDAR-aware AI feedback
- AI confidence score and grade band
- Faculty manual override and printable review pages
- CSV export and curriculum insights
- Image OCR hook using pytesseract + Pillow

Environment variables:
- SECRET_KEY
- DATABASE_URL
- OPENAI_API_KEY
- OPENAI_MODEL (optional)

Note:
Image OCR requires Tesseract OCR binary to exist on deployment.
